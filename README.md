# Projekt2
Projekt 2 realizuje wizualizację procesu przepływu cieczy między zbiornikami, za pomocą rur, oraz dodatkowych elementów: pompy i grzałki. Widoczne są również wykryte alarmy i wykres trendów zmiany parametrów zbiorników. Proces działa automatycznie, a przed jego rozpoczęciem użytkownik zadaje parametry początkowe.
